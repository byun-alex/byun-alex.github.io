# byun-alex.github.io

My personal site: [byun-alex.github.io](https://byun-alex.github.io)

A single hand-written page that presents my projects the way products get launched: one idea per screen, a real screenshot, the skills each project proves, and an honest paragraph on what it actually took to build. Plus a dated build log and case studies of the private systems that run my day.

## Stack

None, on purpose. Hand-written HTML, CSS and one small vanilla JS file for scroll reveals. No framework, no build step, no dependencies, no analytics. Served by GitHub Pages straight from this repo.

## Maintenance

The site was built with Claude (Fable 5) and is maintained the same way. [MAINTAINING.md](MAINTAINING.md) is the full handbook: design philosophy, file map, copy-paste recipes for new sections, voice guide, and anti-drift rules. It is written so that a fresh AI session (or a fresh human) can make a correct edit without any other context.

## Run locally

Open `index.html` in a browser. That is the whole setup.
